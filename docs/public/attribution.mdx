# Attribution

## Original Project

Claude-Mem was originally created by **[@thedotmack](https://github.com/thedotmack)** and is available at:

- **Repository**: [github.com/thedotmack/claude-mem](https://github.com/thedotmack/claude-mem)
- **Documentation**: [docs.claude-mem.ai](https://docs.claude-mem.ai)

We are grateful to thedotmack for creating this excellent tool and making it open source.

## This Fork

This version is maintained by **[Customable](https://customable.de)** and includes the following extensions:

### Additional Features

- **Qdrant Vector Database**: Native TypeScript Qdrant client as alternative to Chroma
- **OpenCode Platform Adapter**: Support for OpenCode IDE integration
- **Enhanced Web UI**: Console logs viewer with filtering capabilities
- **Export/Import API**: Backup and restore endpoints (`/api/export`, `/api/import`)
- **Network Binding**: `CLAUDE_MEM_WORKER_BIND` setting for network exposure
- **Mistral Provider**: Support for Mistral AI models via direct API

### Bug Fixes

- Fixed various queue processing issues
- Improved database migration handling
- Enhanced error logging and debugging
